# Backend - Employee Management API

This is the backend API for the project.

## Details
- Framework: Express.js
- Database: MongoDB via Mongoose
- Deployment: Vercel

## Endpoints
- GET /emp-api/employees: gets the list of employees
- POST /emp-api/employees: adds a new employee
- GET /emp-api/employees/:id: gets one employee by id
- PUT /emp-api/employees/:id: updates an employee
- DELETE /emp-api/employees/:id: deletes an employee
