# Employee Management System

This is a full stack web application to manage employee data. I built this using the MERN stack to learn how to connect a frontend with a database.

## Technologies used
- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB Atlas
- State Management: Zustand
- Deployment: Vercel

## Features
- Add new employees with their name, email, and other details.
- See a list of all employees in a table.
- View specific details of an employee.
- Edit and update employee information.
- Delete employees from the database.

## Setup and Installation

### Backend
1. Go to the Employee-management-project/backend-3 folder.
2. Run npm install to install packages.
3. Create a .env file and add your MongoDB connection string as DB_URL.
4. Run npm start to run the server.

### Frontend
1. Go to the Employee-management-project/frontend-3 folder.
2. Run npm install.
3. Create a .env file and add VITE_API_URL=http://localhost:6001.
4. Run npm run dev to start the application.
